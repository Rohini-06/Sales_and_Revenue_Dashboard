# 📊 Sales and Revenue Dashboard

## 📌 Project Overview
The Sales and Revenue Dashboard is an interactive Power BI project designed to analyze sales performance, revenue trends, product sales, and shipment details. It helps users gain business insights through dynamic visualizations and KPI tracking.

## 🎯 Objectives
- Analyze overall sales and revenue.
- Monitor product performance.
- Compare sales across different countries.
- Track boxes shipped.
- Identify top-performing sales persons.
- Enable interactive filtering using slicers.

## 📂 Dataset
- Source: CSV File
- Total Records: 1,094+
- Fields:
  - Sales Person
  - Country
  - Product
  - Date
  - Amount
  - Boxes Shipped

## 📈 Dashboard Features
- ✅ Total Sales Amount
- ✅ Total Boxes Shipped
- ✅ Total Sales Persons
- ✅ Total Products
- ✅ Sales Trend by Date
- ✅ Sales by Country
- ✅ Sales by Product
- ✅ Top 10 Sales Persons
- ✅ Boxes Shipped by Country
- ✅ Interactive Slicers (Date, Country, Product, Sales Person)

## 🛠️ Tools Used
- Microsoft Power BI Desktop
- Microsoft Excel / CSV
- Git & GitHub


## 🚀 How to Use
1. Download the `.pbix` file.
2. Open it using Microsoft Power BI Desktop.
3. Explore the dashboard using the available slicers and filters.
